Aktau-router is a website designed to provide information, navigation, and local resources for the city of Aktau. It helps users explore key locations, services, and points of interest in the city through a clean and user-friendly interface.

Features
- City Navigation: Easily find locations, neighborhoods, and landmarks in Aktau.
- Interactive Map: Browse the city through an interactive map.
- Local Services: Quick access to local businesses, cafes, and service centers.
- Responsive Design: Works seamlessly on desktops, tablets, and mobile devices.

Installation:
To run the website locally, follow these steps:
- Clone the repository:
https://github.com/asylomir/schoolproject.git
- Open index.html in your browser

Technologies Used:
HTML, CSS, Google Maps API
